# Opis wykonania strony WWW

## Struktura projektu

Projekt składa się z kilku głównych folderów i plików:

- `index.html`: Strona główna zawierająca wprowadzenie i bibliotekę poradników.
- `contents/`: Folder zawierający poszczególne strony z poradnikami.
- `css/`: Folder zawierający pliki CSS do stylizacji strony.
- `js/`: Folder zawierający pliki JavaScript do obsługi interakcji na stronie.
- `images/`: Folder zawierający obrazy używane na stronie.

## Tworzenie stron HTML

Każda strona HTML została stworzona z następującymi elementami:

1. **Nagłówek (`<header>`)**: Zawiera logo, pole wyszukiwania i zegar.
2. **Główna zawartość (`<main>`)**: Zawiera sekcje z treścią poradników.
3. **Stopka (`<footer>`)**: Zawiera podziękowanie za odwiedzenie strony i link do strony głównej.

Przykład struktury strony HTML:
```html
<!DOCTYPE html>
<html lang="pl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tytuł strony</title>
    <link rel="stylesheet" href="../css/styles.css">
</head>
<body>
    <header id="header">
        <a href="../index.html"><img src="../images/logo.png" alt="Logo" class="logo"></a>
        <div class="search-container">
            <input type="text" placeholder="Szukaj...">
            <div class="search-icon">🔍</div>
        </div>
        <div class="clock" id="clock">🕒</div>
    </header>
    <main>
        <!-- Zawartość strony -->
    </main>
    <footer id="footer">
        Dziękujemy za odwiedzenie naszej strony! 😊 Kliknij logo, aby wrócić do indexu! 🏠
    </footer>
    <script src="../js/clock.js"></script>
    <script src="../js/header.js"></script>
    <script src="../js/search.js"></script>
</body>
</html>
```

## Stylizacja CSS

Stylizacja strony została wykonana za pomocą pliku `styles.css`, który zawiera style dla różnych elementów strony, takich jak nagłówki, sekcje, przyciski, tabele itp.

Przykład stylizacji:
```css
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f4f4f9;
}

header {
    background: linear-gradient(90deg, #e0eafc, #cfdef3, #e0eafc);
    color: #333;
    padding: 20px;
    display: flex;
    align-items: center;
    justify-content: space-between;
    position: fixed;
    top: 0;
    width: 100%;
    z-index: 1000;
    transition: transform 0.3s ease-in-out;
}
```

## Skrypty JavaScript

Skrypty JavaScript zostały użyte do obsługi interakcji na stronie, takich jak zegar, wyszukiwanie i kalkulator RAM.

Przykład skryptu:
```javascript
// filepath: /c:/Users/Admin/Desktop/WWWcosidk-14-18/js/clock.js
function updateClock() {
    const clockElement = document.getElementById('clock');
    const now = new Date();
    clockElement.textContent = now.toLocaleTimeString();
}

setInterval(updateClock, 1000);
updateClock();
```

## Podsumowanie

Strona została stworzona z myślą o łatwej nawigacji i dostępie do poradników dotyczących tworzenia i zarządzania serwerem Minecraft. Każda strona zawiera szczegółowe instrukcje i przykłady kodu, które pomagają użytkownikom w konfiguracji i optymalizacji serwera.

# Opis procesu tworzenia strony WWW

## Temat strony

Tematem strony jest poradnik dotyczący tworzenia i zarządzania serwerem Minecraft. Strona spełnia cztery warunki:
1. **Obszerną treść tekstową**: Strona zawiera szczegółowe instrukcje dotyczące różnych aspektów zarządzania serwerem Minecraft, takich jak podstawy, instalacja pluginów, administracja, hosting, konfiguracja, optymalizacja i najlepsze pluginy dla trybu survival.
2. **Różnorodną treść multimedialną**: Strona zawiera obrazy, które ilustrują różne sekcje poradnika.
3. **Zagadnienia obliczeniowe**: Strona zawiera skrypty JavaScript, które realizują różne funkcje, takie jak wyszukiwanie treści oraz kalkulator RAM.
4. **Ciekawe skrypty Java pozyskane z otwartych źródeł internetowych**: Skrypty JavaScript zostały pozyskane z otwartych źródeł internetowych i w przemyślany sposób wykorzystane na stronie.

## Struktura strony

Strona została podzielona na wiele plików, aby spis treści był w pliku `index.html`, a poszczególne rozdziały w oddzielnych plikach:
- `index.html`: Spis treści i główna strona poradnika.
- `contents/instalacja-serwera.html`: Podstawy tworzenia serwera Minecraft.
- `contents/pluginy-i-modyfikacje.html`: Najlepsze pluginy dla serwera Minecraft.
- `contents/administracja-i-konfiguracja-serwera.html`: Administracja serwera Minecraft.
- `contents/hosting-i-domena.html`: Jak hostować serwer Minecraft.
- `contents/ekonomia-i-mechaniki-gry.html`: Konfiguracja plików serwera Minecraft.
- `contents/minigry-i-tryby-gry.html`: Najlepsze pluginy dla trybu survival.
- `contents/optymalizacja-i-bezpieczenstwo.html`: Optymalizacja wydajności serwera Minecraft.
- `contents/problemy-i-rozwiazania.html`: Rozwiązywanie problemów z serwerem Minecraft.
- `contents/bonus-poradniki-i-spolecznosc.html`: Dodatkowe poradniki i społeczność.

## Podkatalogi

Dla lepszego uporządkowania utworzono odrębne podkatalogi dla obrazków i treści tekstowej:
- `images/`: Katalog zawierający obrazy używane na stronie.
- `contents/`: Katalog zawierający pliki HTML z poszczególnymi rozdziałami poradnika.

## Nawigacja

Strona zawiera alternatywne sposoby poruszania się po stronie, takie jak:
- **Spis treści**: Główna nawigacja w pliku `index.html`.
- **Linki między treściami powiązanymi**: Linki w poszczególnych rozdziałach prowadzące do innych powiązanych sekcji.

## Style CSS

Do formatowania strony użyto stylów CSS w odrębnym pliku `styles.css`. Style te obejmują:
- Ogólne formatowanie strony.
- Stylizację nagłówków, sekcji, list i kodu.
- Stylizację paska wyszukiwania.
- Efekty animacji, takie jak fade-in dla treści po załadowaniu strony.

## Skrypty JavaScript

Skrypty JavaScript zostały wydzielone do plików typu `.js`:
- `js/search.js`: Skrypt realizujący funkcję wyszukiwania treści na stronie.
- `js/clock.js`: Skrypt realizujący aktualizację zegara na stronie.
- `js/ram-calculator.js`: Skrypt realizujący obliczanie RAM na podstawie liczby graczy i pluginów.

## Inspiracje i źródła

Inspiracje do skryptów JavaScript zostały pozyskane z otwartych źródeł internetowych, takich jak:
- **MDN Web Docs**: Dokumentacja i przykłady dotyczące JavaScript i CSS.
- **Stack Overflow**: Forum z pytaniami i odpowiedziami dotyczącymi programowania.
